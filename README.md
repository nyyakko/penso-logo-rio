# Penso, Logo Rio

Gosta de piadas de excelentíssima qualidade? Pois está no lugar errado.

## Como executar o projeto localmente

### Configurando o ambiente

Primeiramente, inicie o projeto com o seguinte comando:

```bash
$ npm install
```

### Executando

Após isso, simplesmente execute-o da seguinte maneira:

```bash
$ npm run web
```

se tudo estiver correto, o projeto deverá iniciar sua execução em http://localhost:8081.
